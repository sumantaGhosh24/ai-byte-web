# AIByte Landing Page

Production-grade landring page and dashboard for **AIByte**, an Expo micro-learning platform focused on AI learning, video lessons, quizzes, progress tracking, streaks, notifications, and scalable retention systems.

Built with:

- Node.js
- Express.js
- TypeScript
- PostgreSQL
- Drizzle ORM
- Clerk Authentication
- Redis
- Neon DB
- Zod Validation
- Sentry
- Arcjet
- Rate Limiting
